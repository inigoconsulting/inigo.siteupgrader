inigo.siteupgrader Installation
-------------------------------

To install inigo.siteupgrader using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``inigo.siteupgrader`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        inigo.siteupgrader

* Re-run buildout, e.g. with:

    $ ./bin/buildout
